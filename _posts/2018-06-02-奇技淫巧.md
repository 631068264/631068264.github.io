# 百度云盘播放加速 Chrome

在 console 中
2 倍加速
```js
videojs.getPlayers("video-player").html5player.tech_.setPlaybackRate(2)
```
还原
```js
videojs.getPlayers("video-player").html5player.tech_.setPlaybackRate(1)
```
